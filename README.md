# MojaBeseda 1.3

Nadgradnja vključuje:
- novo DG Smart app ikono,
- odstranjen podvojeni napis in hitre gumbe,
- popravljeno Google prijavo s popup/redirect rezervnim načinom,
- sinhronizacijo poti in dnevnika,
- delujoče Moje poti,
- iskanje dnevne spodbude po ključni besedi,
- sistemsko zvočno branje,
- kartico O aplikaciji,
- Nadaljuj branje.
